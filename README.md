# Customer-Segmentation
Customer Segmentation helps in analyzing customer behavior based on their buying habits, and it helps in target business strategies. It compares between the income and spending score of the customers to find the clusters, i.e. , different customer segments are identified. 

Objective:
To analyze customer behavior and group them into segments using machine learning.

Tools Used:
Python
Pandas
Matplotlib
Scikit-learn

Steps:
Data analysis
Visualization
Elbow method
K-Means clustering

Results:
Identified 5 customer segments
Found patterns in income vs spending

Insights:
Premium customers exist
Some high-income users spend less → potential targets
